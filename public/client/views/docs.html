WEBB-API

MovieBox är en mashup-tjänst baserat på tre olika APIer som erbjuder ett REST-API för de utvecklare som vill skapa en applikation baserat på film och tillhörande snacks.

Metoder

GET/showSnacks

Denna funktion visar all data som hämtas ifrån matapi.se/foodstuff, denna data kan användas till att rendera ut lämpligt snacks till en specifik film. Det kan göras genom input av ett ID som varje snacks har.

Exempel:

Utan input av id:

{ "name": "Talg nöt", "number": 1 }, { "name": "Späck gris", "number": 2 }, { "name": "Ister gris", "number": 3 }, { "name": "Kokosfett", "number": 4 }, { "name": "Matfettsblandning havssaltat fett 80% berikad typ Bregott", "number": 5

Med input av ID:

Input-id: 1864

http://localhost:3000/showSnacks/1864

{Karameller syrliga}

GET/showMovie

Denna funktion visar upp en slumpmässigt vald film med tillhörande information. Kan användas som utvecklaren vill skapa en applikation där den visar upp olika filmval.

Exempel:

{ "youtubeId": "RoLYZf-1E8o", "movieInfo": { "Title": "Home", "Year": "2009", "Genre": "Documentary, Drama, Family", "Plot": "With aerial footage from fifty-four countries, 'Home' is a depiction of how Earth's problems are all interlinked." } }
