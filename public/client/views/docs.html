MOVIEBOX API

Moviebox API är till för utvecklare som vill hämta ut en slumpad film. Samt för de utvecklare som vill hämta ut matdata eller ett specifikt matobjekt.

Klicka här för att se MovieBox API via apiary.io

[GET]/getSnacks

Lista all mat

Response 200 (application/json)

                        
                            [
                            {
                            "name": "rågmjöl",
                            "number": 1
                            }, {
                            "name": "vetemjöl",
                            "number": 2
                            }, {
                            "name": "popcorn",
                            "number": 3
                            }, {
                            "name": "rostade saltade kikärtor",
                            "number": 4
                            }
                            ]

[GET] /getSnacks/:snack

Listar specifikt snacks

Går du till ändpunkten /getSnacks/ och lägger till ett id som exempelvis 1 (/getSnacks/1). Så får du svar med det specifika id-namnet.

Response 200 (application/json)

                        
                        [
                        {
                        "name": name
                        }
                        ]
